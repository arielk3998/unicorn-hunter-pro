## ğŸ¨ ELITE Framework (Resume Organization)

### Section Priority Order

**For Technical/Engineering Roles**:
1. **Technical Skills** (highlight immediately)
2. **Professional Experience** (STAR bullets)
3. **Key Achievements** (metrics-heavy)
4. **Education** (unless recent grad)
5. **Certifications** (if relevant)

**For Leadership/Management Roles**:
1. **Professional Summary** (leadership focus)
2. **Leadership Experience** (team size, scope)
3. **Key Achievements** (business impact)
4. **Technical Skills** (supporting)
5. **Education**

**For Career Changers**:
1. **Transferable Skills**
2. **Relevant Projects** (even if unpaid)
3. **Professional Experience** (framed for new field)
4. **Education/Certifications** (recent training)

âœ… **DO**:
- Use standard fonts (Arial, Calibri, Times New Roman)
- Use standard section headings (PROFESSIONAL EXPERIENCE, EDUCATION)
- Include keywords from job description

âŒ **DON'T**:
